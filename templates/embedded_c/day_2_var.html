Hello, World!

Every programming journey starts with the infamous "Hello, World!" program. Ours is not going to be different.



Hello, World! Program

Open any of the IDEs described in the previous chapter or any IDE you might be familiar with. You could also use the compiler to the right to execute your code. Thanks to JDoodle for this wonderful online compiler. Great job!

  1. Using the compiler on the right enter the following code
  2. Toggle the "interactive button" on the compiler to begin editing the code.
  3. Type "#include " followed by two line spaces
  4. Type "int main()" followed by opening curly bracket "{", space in-between and then "}".
  5. Then type "printf("Hello, world!");"
  6. Then type "return 0;"


Structure of a C Program

C program basically consists of the following parts

- Preprocessor Commands
Inbuilt
Userdefined
- Functions: The "int main() {}" is an example of a function. "printf()" is also an example of a function. The "Hello, world!" is passed into the function and it is printed on the screen.
- Variables: There are no variables in this example but that will be discussed soon.
- Statements & Expressions: A statement is an instruction to a computer to perform a task. An example of an expression is "printf("Hello, world!");". The instruction is that the computer should print "Hello, world!" to the screen. A statement is succeeded by a semicolon.
- Comments: Comments are necessary in the good programming for proper documentation and maintenance.



Code Execution

Exercise

  1. Add a second printf() statement to the main() function. In the second printf() statement, add "This is my second program." Remember to add a semicolon(;) at the end. Observe the output.
  2. Try adding a "\n" after "Hello, world!" like this "Hello, world! \n". Observe the output. The combination of the backward slash(\) and a character such as "n" is called an escape sequence.
  3. Try changing the words "Hello, world!" to something different and hit execute button. Practice and have fun.
